# Portfolio v2 — Phase 0: Foundations Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Stand up an isolated, statically-exported Next.js 15 + TypeScript app under `v2/` with a validated, build-time data pipeline sourced from v1's canonical JSON, plus the unit/e2e test harness — a deployable foundation that renders real data counts.

**Architecture:** A Next.js App Router app in `v2/` (separate from the untouched Flask v1 at repo root). The 4 canonical JSON files in `static/data/` stay the single source of truth; a `tsx`-run sync script copies them into `v2/data/` and validates them against shared Zod schemas at `predev`/`prebuild`/`pretest`, failing fast on schema drift. Typed loaders import the synced JSON and re-parse with the same schemas, so every consumer gets runtime-validated, statically-typed data. `output: 'export'` produces a fully static site (no server, no cold starts).

**Tech Stack:** Next.js 15 (App Router, React 19), TypeScript, Tailwind CSS v4, Zod, tsx, Vitest + @testing-library/react + jsdom, Playwright, ESLint, Prettier.

- [ ] **Step 2: Implement `v2/scripts/sync-data.ts`**

```ts
import { fileURLToPath } from "node:url";
import path from "node:path";
import fs from "node:fs";
import {
  projectsSchema,
  coursesSchema,
  researchListSchema,
  certificationsFileSchema,
} from "../lib/schemas";

const here = path.dirname(fileURLToPath(import.meta.url)); // v2/scripts
const SRC = path.resolve(here, "../../static/data"); // repo-root/static/data
const DEST = path.resolve(here, "../data"); // v2/data

const FILES = [
  { name: "projects.json", schema: projectsSchema },
  { name: "courses.json", schema: coursesSchema },
  { name: "research.json", schema: researchListSchema },
  { name: "certifications.json", schema: certificationsFileSchema },
] as const;

function main() {
  if (!fs.existsSync(SRC)) {
    console.error(`[sync-data] canonical data dir not found: ${SRC}`);
    process.exit(1);
  }
  fs.mkdirSync(DEST, { recursive: true });

  for (const { name, schema } of FILES) {
    const srcPath = path.join(SRC, name);
    const raw = fs.readFileSync(srcPath, "utf8");
    const parsed = JSON.parse(raw);
    const result = schema.safeParse(parsed);
    if (!result.success) {
      console.error(`[sync-data] SCHEMA DRIFT in ${name}:`);
      console.error(JSON.stringify(result.error.issues, null, 2));
      process.exit(1);
    }
    fs.writeFileSync(path.join(DEST, name), JSON.stringify(parsed, null, 2));
    console.log(`[sync-data] ok: ${name}`);
  }
  console.log("[sync-data] all data synced + validated");
}

main();
```
