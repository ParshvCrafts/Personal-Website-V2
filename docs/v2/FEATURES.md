# Features

## Core experience

- Sticky, section-based single page portfolio.
- Scroll-aware navigation with active section state.
- Theme switcher with four palettes.
- Premium hero with stronger typography and composition.
- Scroll-driven showcase section for high-end motion.

## Content sections

- Hero
- About
- Academics
- Research
- Terminal
- Journey
- Skills
- Certifications
- Professional Development
- Projects
- Hobbies
- Contact
- Footer

## Motion system

- GSAP-based reveal and scroll choreography.
- ScrollTrigger for pinning, scrub, and staged storytelling.
- Reduced-motion fallback that removes non-essential motion.
- Small reusable motion primitives instead of one-off animation code.

## Design system

- Four semantic palettes: Midnight, Daylight, Manuscript, Neon (futuristic/tech).
- Editorial display typography paired with a readable body font.
- Consistent spacing, radius, and elevation tokens.
- Grid-led layouts with asymmetric accents.
- Visual hierarchy driven by type, space, and contrast.

## SEO and metadata

- Canonical URL set to `https://www.portfolio.parshvpatel.com/`.
- Open Graph and Twitter cards.
- Person JSON-LD.
- Sitemap and robots support.
- Favicon set and share image.

## Accessibility

- Keyboard complete.
- Visible focus states.
- 44 by 44 minimum touch targets.
- Contrast checked per theme.
- Motion opt-out respected.

## Data contract

- Projects, courses, research, and certifications remain JSON-driven.
- Typed v2-only content can live in small source modules.
- Build-time validation should fail on invalid or drifting data.

## Asset requests for the user

- Hero portrait at 1200 px or larger.
- OG image at 1200x630.
- Higher-res project images if the current assets are low quality.
- Optional sequence frames or prompt-based generated art for the scroll showpiece.
