# Design Direction

## Overall feel

The v2 should feel like an editorial portfolio built by a senior designer: calm, high-contrast, deliberate, and expensive-looking.

## What to avoid

- Generic glassmorphism everywhere.
- Random neon gradients.
- Emoji-driven iconography.
- Over-animated pages.
- Centered single-column templates that feel like startup landing pages.
- Purple-default palettes and stock Inter-on-white layouts.

## Visual principles

- Use large display type with tighter vertical rhythm.
- Let whitespace do more work.
- Keep one strong accent per theme.
- Use asymmetry and section contrast to create movement.
- Make cards feel curated, not boxed-in.

## Typography

- Use a distinctive display face for hero and major headings.
- Use a clear, modern body font for readability.
- Use mono only for labels, code, dates, and terminal surfaces.
- Avoid default system stacks unless there is a fallback requirement.

## Palette guidance

- Midnight: dark, technical, sharp, and cool.
- Daylight: recruiter-safe, bright, and corporate-friendly.
- Manuscript: warm, literary, and more creative.
- Neon: futuristic/tech — space-navy base, electric cyan + neon purple, crisp white.

## Motion guidance

- Use motion to reveal hierarchy, not to decorate every edge.
- Prefer a few strong reveals, scroll pins, and thoughtful transitions.
- Keep hover motion subtle.
- Never make motion required to understand the page.

## Layout guidance

- Use a strong max width and a consistent content grid.
- Alternate dense and open sections to keep pacing interesting.
- Reserve the heaviest motion for the hero and one signature showpiece.
- Use bento-style composition for projects and featured highlights.

## Image guidance

- Use images that support the story, not just fill space.
- Portraits should be clean, high resolution, and easy to crop.
- Project covers should use consistent aspect ratios when possible.
- Any generated sequence should be compressed and budgeted carefully.

## Review checklist

- Does it still look custom at a glance?
- Does the typography feel intentional?
- Are theme colors still accessible?
- Is the motion readable rather than noisy?
- Would this still look good if all animations were disabled?
